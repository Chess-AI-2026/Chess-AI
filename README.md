SP-14-Blue-Chess AI README:

SP-14-Blue-Chess AI is an interactive chess application where you can play against an AI agent. There are 3 difficulties Easy, Medium, Hard. The AI uses the Minimax algorithm with Alpha Beta Pruning to evaluate board positions and select the best move. It is made using python.

Game Modes:
Local PvP: On the same device you can go against some else.
AI - Easy: Depth 1, looks 1 move ahead and has a random chance to make a 'mistake'
AI - Medium: Depth 2, looks 2 moves ahead and has a low chance to make a 'mistake'
AI - Hard: Depth 3, looks 3 moves ahead and makes no 'mistakes'
Quit: Closes the program

Requirements:
Python 3.13
Must have pygame installed
Must run main.py (located in the src folder)

Controls:
Click and drag pieces to move
'R' key resets the game
'T' key changes the color of the board
'ESC' key pauses the game
